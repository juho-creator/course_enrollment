# 选课宏
适用于课程选课的罗技 G502 LUA 脚本
</br></br>

## 演示
[![图像](http://i3.ytimg.com/vi/UmjRJE3Npoc/hqdefault.jpg)
](https://www.youtube.com/watch?v=UmjRJE3Npoc)
</br></br>


## 用户指南
1. 将 LUA 脚本加载到罗技 G502 上
2. 按下中间按钮获取课程选课按钮的位置
3. 右键单击即可立即移动到课程选课按钮。
